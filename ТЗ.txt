Общая идея состоит в создании сервиса, который собирает информацию о валютах, как-то классифицирует ее.
На сайте можно авторизоваться и получить допонтительные возможности: изменение стоимости валюты за последние 5 дней,
редактирование личного кабинета (пользователь может выбрать только нужные ему позиции), а также выгрузка данных в Excel файл


У бота реализован следующий функционал:
"Валюта" отвечает за получение различной информации о курсе валют:
    "Текущий курс" показывает список всех валют и их текущий курс
    "Выбрать валюту" показывает данные о какой-либо валюте за выбранный период

"Акции" показывает данные о выбранной акции за выбранный период. аналогично действию "Выбор валюты"

"Организации" ищет ближайшие банки, банкоматы или пункты обмена валют и информацию о них.
              пользователь выбирает тип организации и местоположение

"Рассылка" сообщает пользователю, когда курс какой-либо валюты вырастет или понизится на более чем на р процентов
    "Добавить" добавление новой подписки пользователем. Пользователь выберает валюту, период,
               за который нужно смотреть изменение (день или неделя), и сам процент изменения
    "Отписаться от всех" удалит все подписки пользователя на рассылку
    "Отписаться от одной" показывает список всех подписок пользователя с возможностью удаления одной или нескольких из них


Про запуск
Переменные окружения:
    HOST
    SECRET_KEY
    TOKEN_VK - ключ доступа к vk api
    GROUP_ID - id сообщества вк
    IMAGE_ID - id изображения "stonks.jpg" (изображение должно быть загружено в основной альбом группы)
    API_KEY - ключ доступа к api поиска по организациям
